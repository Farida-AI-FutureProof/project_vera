# Data Directory

This folder stores the demonstration datasets used for Project VERA validation and PoC.

Organize the datasets into the following subfolders:

- `aviation` – Regulatory incident reports (e.g., NTSB Aviation Accident Database).
- `fda` – Enforcement and recall reports (e.g., FDA Enforcement Reports).
- `logistics` – Supply chain and logistics operations data.
- `synthetic_emails` – Synthetic internal emails and change approvals for discrepancy detection and RBAC testing.

Each subfolder should contain the processed documents and a README describing the source and any preprocessing steps.
